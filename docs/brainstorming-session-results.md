## Técnicas Utilizadas

### 1. Word Association (Warm-up)

**Objetivo:** Soltar la mente, generar material base

| Palabra estímulo | Asociaciones generadas |
|------------------|----------------------|
| TERMINAL | place, unique, center, orchestration |
| CLAUDE | AI, Assistant, dev |
| FLOW | workflow, organization, multiplex |
| POWER | unlimited, multi, organization |
| CONTROL | center, just in one, operator |
| BRIDGE | link, easy, union |

**Patrones identificados:**
- "center" apareció 2 veces
- "organization" apareció 2 veces
- Tendencia hacia conceptos de control centralizado

---

### 2. Metáforas y Analogías

**Objetivo:** Encontrar conceptos abstractos y memorables

| Pregunta | Respuesta |
|----------|-----------|
| "Mi app es como un ______ para desarrolladores" | **cockpit** |
| "Si mi app fuera un lugar físico, sería un ______" | **torre de control** |
| "Mi app hace con Claude lo que un ______ hace con ______" | **piloto de avión hace con la consola** |
| "El usuario de mi app se siente como un ______" | **arquitecto** |

**Insight clave:** Metáfora de AVIACIÓN resonó fuertemente (cockpit, torre de control, piloto, consola)

**Exploración profunda de metáforas:**

| Categoría | Palabras generadas |
|-----------|-------------------|
| Cockpit/Aviación | throttle |
| Términos aviación cool | vector |
| Arquitectura | scaffold, foundation |

---

### 3. Mashups/Fusiones

**Objetivo:** Combinar palabras para crear nombres únicos

**Combinaciones generadas:**
- cockpit + port = CockpitPort
- cockpit + nest = CockpitNest
- cockpit + hub = CockpitHub
- term + cockpit = **TermCockpit** (favorito)

**Variaciones evaluadas:**

| Nombre | Reacción |
|--------|----------|
| Cockpit (solo) | Pensar |
| CodeCockpit | Buena |
| DevCockpit | Buena |
| Cockpit.dev | Buena |
| CockpitAI | Mala |
| MyCockpit | Mala |
| TheCockpit | Mala |
| TermCockpit | Pro/Favorito |

**Patrón:** Prefijos genéricos (My, The) y sufijo AI rechazados

### Conflicto identificado

**Cockpit Project** (cockpit-project.org) es una herramienta web existente para administrar servidores Linux. Por eso se descartó "Cockpit.dev" solo y se prefirió "TermCockpit" que es único.

## Reflexión de la Sesión

### Lo que funcionó bien
- Word Association como warm-up efectivo
- Metáforas revelaron patrón de aviación rápidamente
- Mashups permitieron refinar hacia nombre final

### Insight clave
La metáfora de **cockpit/aviación** resonó fuertemente porque captura la esencia de la app: control centralizado, profesional, piloto al mando de una herramienta poderosa.

### Nombre elegido: TermCockpit

**Significado:** Terminal + Cockpit = El cockpit donde controlas tu terminal y Claude

**Tagline potencial:** "Your AI development cockpit" / "Take control of Claude"
